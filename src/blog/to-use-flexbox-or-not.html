I recently ran into a problem where I needed to order something differently on mobile. The problem occured on this very website, in fact - the blue ribbon on the blog page.

If you make websites you know this can be problematic. When using a grid framework it can be done with column pushes and pulls — but those feel hacky. You can also change up your HTML source order and maybe also some CSS rules for that single component. Or you can use Flexbox.

For me, I'm not too worried about browser coverage here. But, keep in mind, if a good portion of your users are stuck on IE9 you should probably reconsider using Flexbox.

In my case, a little Flexbox fixed everything. 

  .blog-posts-container{
    display: flex;
    flex-wrap: wrap;
    .above-fold{
      order: 2;
    }
  }
  .blog-posts-ribbon{
    order: 1;
  }
